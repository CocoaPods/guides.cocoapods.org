Maintaining & Best Practices

This guide gathers best practices for Cocoa library authors. The goal is to make integration of libraries easier for everyone, using CocoaPods or not. Ultimately, CocoaPods will leverage existing Xcode projects to build libraries, so this will require some rigor from library authors.
